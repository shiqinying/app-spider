# app-spider
uiautomator2+mitmdump实现对app的爬取

# 环境
* python3.6
* windows
* 安卓手机

# 使用方法
* 将config_template.py文件更改为config.py,配置相应变量
* 创建虚拟环境
```shell
mkvirtualenv env_name --python3.6
workon env_name
```
* 安装依赖包 pip install -r requirements.txt
